# Platform/core
Thư viện hỗ trợ trong qúa trình phát triển ứng dụng

`support`: Chứa những class hỗ trợ: Query, Event, Request api ...
`setting`: Hỗ trợ load plugins
`binder-proxy`: Hỗ trợ bind-proxy.
`base`: Chứa các class dùng chung.
